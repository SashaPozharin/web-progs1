Основы JS
Переменные (Variables)
Типы переменных
Int - целые числа (74 69)
Float - не целые числа (8,9 1,3)
char - только одна буква ('a', ,'s', "4", "!" - это символы)
string - тоже самое, как char, но можно использовать неограниченное количество знаков
("asd", 'sudo', "!!@@@$##" - это символы)
bool - true, false

Обьявление переменных (Declaration ffof Variables)
let NAME = INITIAL_VALUE;
let username = "";
let userage = 15;

Правило названий переменных (naming rules)
1) переменные называются с маленькой буквы
2) название должно быть логическим и обоснованным
3) если название состоит из нескольких слов, то должно быть оформлено в виде Горба Верблюда
Let nameLenght является правильным

Начальные значения (INITIAL_VALUE)
Начальные значения для чисел - 0 или то число, которое должно быть в начале
Строка/символ - "" или '' или "Начальное_ЗНАЧЕНИЕ" ("Степашка")
bool - true/false
УНИфИЦИРОВАННЫЙ - "" или null
